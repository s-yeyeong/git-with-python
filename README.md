# 🎵 나만의 K-pop 퀴즈 게임

파이썬으로 만든 터미널 기반의 퀴즈 게임입니다. K-pop을 주제로 한 퀴즈를 풀고, 나만의 퀴즈를 추가할 수 있습니다.

## 🌟 주요 기능
1. **퀴즈 풀기**: 등록된 퀴즈를 풀고 정답을 맞히면 점수가 올라갑니다.
2. **퀴즈 추가**: 사용자 입력을 통해 새로운 퀴즈를 직접 추가합니다. (빈칸, 문자, 범위 밖 입력 등 예외 처리 완벽 방어!)
3. **퀴즈 목록**: 현재 등록된 모든 퀴즈 리스트를 확인합니다.
4. **점수 확인**: 역대 최고 점수를 보여줍니다.
5. **데이터 영구 저장**: `state.json` 파일을 이용해 게임을 껐다 켜도 퀴즈와 점수가 그대로 유지됩니다.
6. **안전 종료**: 게임 도중 `Ctrl+C`로 강제 종료해도 데이터가 날아가지 않게 안전하게 저장하고 종료됩니다.

## 🛠 사용 기술
- **Language**: Python 3
- **Data Storage**: JSON (`state.json`)
- **Version Control**: Git & GitHub

## 🚀 실행 방법
터미널에서 아래 명령어를 입력하여 프로그램을 실행합니다.
```bash
python main.py
